# Cordova Plugin: Deeplink Handler

This plugin allows Cordova-based Android applications to handle deep links. It integrates deep link handling through intent filters and supports Android App Links with `assetlinks.json` for verification.

## Features

- Handles incoming deep links for your Cordova app.
- Supports Android App Links verification with `assetlinks.json`.
- Extracts deep link URLs for navigation within the app.

## Installation

Install the plugin into your Cordova project:

```bash
cordova plugin add path/to/cordova-plugin-deeplink-handler
